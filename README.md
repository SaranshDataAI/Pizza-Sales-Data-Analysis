# Pizza Sales Data Analysis (SQL + Power BI)

## Project Overview
This project analyzes pizza sales data to identify key business insights such as revenue trends, customer ordering behavior, and product performance.

## Tools Used
- SQL
- Power BI
- Data Visualization

## Key KPIs
- Total Revenue
- Average Order Value
- Total Orders
- Total Pizzas Sold
- Average Pizzas Per Order

## Business Insights
- Highest order days: Wednesday and Thursday evenings
- Peak months: April and January
- Classic category contributes the highest sales
- Large size pizzas generate the most revenue

## Dashboard
Power BI interactive dashboard showing sales trends and product performance.
